How to Download fonts locally

1. check out git@github.com:hlasith/fontselect-jquery-plugin.git

2. goto /get-latest-google-fonts-bash.php

3. select the list by popularity and submit

4. select the number of font you need from the text area.

5. goto /google-font-download and edit google-font-download-popular-script.sh 

6. copy selected fonts to google-font-download-popular-script.sh

7. run google-font-download-popular-script.sh

8. popular google fonts will download.
